# weather-app

Please create '.env.local' file in app root folder,
And put your google and cwb api key in it.

```js
VUE_APP_CWB_API_KEY={ your-cwb-api-key }
VUE_APP_GOOGLE_API_KEY={ your-google-geocoding-api-key }
```

## Project setup

```js
npm install
```
